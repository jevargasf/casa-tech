Landing page ficticia desarrollada en mi rol de ayudante como ejercicio para ayudantía de módulo de Front End en el Bootcamp Fullstack Java de Talento Digital.

Se desarrollaron los siguientes puntos:
- Marca ficticia de tienda de productos electrónicos con nombre, logo, paleta de colores y textos descriptivos.
- Landing page simple utilizando estándares de la industria tales como: componentes web estándar que potencien la legibilidad, usabilidad y responsividad de la página.
- Librería Bootstrap 5.3 con estilos personalizados, reescribiendo la paleta de colores acorde con marca empresarial.
- Script simple para desplegar alerta de éxito.
